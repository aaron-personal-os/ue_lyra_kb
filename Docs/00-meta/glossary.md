# 项目术语表

> 记录项目中的术语、缩写、同义词。

## A

### Ability System

参见 GAS（Gameplay Ability System）

### Action

UE Action = UE 编辑器操作 = AI 可调用的 UE 操作

## C

### Component

组件，UE 中可复用的功能模块。

### C++

Unreal Engine 的主要编程语言。

## D

### DataTable

数据表，UE 中用于存储结构化数据的资产类型。

## E

### Experience

体验定义（Experience Definition），Lyra 项目的核心概念，用于定义游戏的完整体验（包括角色、武器、UI 等）。

## G

### GAS

Gameplay Ability System，UE 的游戏能力系统，用于实现技能、属性、效果等。

### GameMode

游戏模式，定义游戏的规则和流程。

### GameState

游戏状态，存储游戏的全局状态信息。

## L

### Lyra

LyraStarterGame，UE5 的官方示例项目。

## M

### Module

模块，UE 代码组织的基本单位。

## P

### PlayerController

玩家控制器，处理玩家输入和控制逻辑。

### Pawn

 pawn，可被控制的 Actor。

## S

### StateTree

状态树，UE 5.x 引入的新行为树系统，用于 AI 决策。

### Subsystem

子系统，UE 中的全局管理系统。

## T

### Tick

帧更新，UE 中的每帧处理逻辑。

## U

### UE

Unreal Engine，虚幻引擎。

### UObject

UE 中所有对象的基类。

### UMG

Unreal Motion Graphics，UE 的 UI 框架。

## W

### Widget

UI 控件，UMG 中的 UI 元素。
