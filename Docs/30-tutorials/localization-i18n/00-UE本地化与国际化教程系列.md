> 从概念到实践，系统掌握 UE 多语言支持的全套机制。

- **篇数**：7 篇
- **难度**：初级 → 中级
- **预计学习时间**：8 小时
- **UE 版本**：5.7

## 学习路线

### 阶段一：基础概念（1-2 小时）

先理解国际化和本地化的基本概念，建立对 UE 本地化系统的整体认识。

→ 从 [00-UE本地化与国际化概览](00-UE本地化与国际化概览.md) 开始

### 阶段二：文本本地化（2-3 小时）

掌握 FText、String Table、文本格式化等核心机制，这是本地化系统的基础。

→ 学习 [02-文本本地化深入FText与StringTables](02-文本本地化深入FText与StringTables.md) 和 [03-本地化仪表盘与工作流](03-本地化仪表盘与工作流.md)

### 阶段三：高级主题与实战（3-4 小时）

学习资产本地化、运行时语言切换，最后通过 Lyra 案例巩固所学知识。

→ 完成 [04-资产本地化音频纹理与多媒体](04-资产本地化音频纹理与多媒体.md) 到 [06-Lyra本地化实践案例](06-Lyra本地化实践案例.md)
