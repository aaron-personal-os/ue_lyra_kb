# 表面采样实战

> **前置知识**：[05-常用 PCG 节点](./05-常用PCG节点详解.md)
> **预计阅读时间**：30 分钟

## 概念直觉

### 从"均匀采样"到"智能采样"

**基础采样**（第 5 课）：
```
均匀密度 → 均匀分布 → 不自然
```

**高级采样**（本课）：
```
密度贴图 + 图层权重 + 法线过滤 → 自然分布
```

**核心思想**：用 **数据** 控制采样，而不是固定参数。

---

## 技术机制

### 1. 密度贴图（Density Texture）

**功能**：用纹理的 **亮度** 控制采样密度。

```
黑色 (RGB=0) → 密度 0（不生成）
白色 (RGB=255) → 密度 1（全生成）
灰色 (RGB=128) → 密度 0.5（半生成）
```

#### 参数配置

| 参数 | 类型 | 说明 |
|-----|------|------|
| `Density Texture` | `TSoftObjectPtr<UTexture2D>` | 密度贴图 |
| `Texture Density Scale` | `float` | 密度缩放（默认 1.0） |
| `bUseTexture Alpha` | `bool` | 是否使用 Alpha 通道 |

#### 使用示例

**目标**：在草地上生成花朵，密度随草地纹理变化。

```
[Surface Sampler]
    Density Texture = T_GrassDensity.png
    Texture Density Scale = 2.0

[Static Mesh Spawner]
    Static Mesh = Flower_01
```

**关键发现**：
- 纹理尺寸影响精度（建议 1024x1024+）
- 使用 `sRGB = false`（线性空间）
- 支持 **运行时生成** 的纹理（Render Target）

---

### 2. 图层权重（Layer Weight）

**功能**：根据 **地形图层** 权重控制采样。

**适用场景**：地形图层（如 Grass、Rock、Snow）。

#### 参数配置

| 参数 | 类型 | 说明 |
|-----|------|------|
| `Layer Name` | `FName` | 地形图层名称 |
| `Layer Weight Threshold` | `float` | 权重阈值 [0, 1] |
| `bInvert Layer` | `bool` | 是否反转权重 |

#### 使用示例

**目标**：只在 `Grass` 图层上生成草。

```
[Surface Sampler]
    Layer Name = "Grass"
    Layer Weight Threshold = 0.5

[Static Mesh Spawner]
    Static Mesh = Grass_01
```

**关键发现**：
- 只适用于 `Landscape`（不支持 Static Mesh）
- 图层名称必须 **完全匹配**（区分大小写）
- `bInvertLayer = true` → 在 **非 Grass** 区域生成

---

### 3. 法线过滤（Normal Filter）

**功能**：根据 **表面法线** 过滤采样点（如只在平坦区域生成）。

#### 参数配置

| 参数 | 类型 | 说明 |
|-----|------|------|
| `bFilter by Normal` | `bool` | 是否启用法线过滤 |
| `Normal Threshold` | `float` | 法线阈值（角度，度） |
| `Reference Normal` | `FVector` | 参考法线（默认 Z 轴） |

#### 原理

**法线**：表面的朝向（单位向量）。

```
平坦地面：Normal = (0, 0, 1)（Z 轴向上）
垂直墙面：Normal = (0, 1, 0)（Y 轴朝外）
```

**过滤逻辑**：
```cpp
// 计算角度
float Angle = FMath::Acos(FVector::DotProduct(Normal, ReferenceNormal));
Angle = FMath::RadiansToDegrees(Angle);

// 过滤
if (Angle > NormalThreshold)
{
    // 剔除这个点
}
```

#### 使用示例

**目标**：只在坡度 < 30° 的区域生成树木。

```
[Surface Sampler]
    bFilter by Normal = true
    Normal Threshold = 30.0
    Reference Normal = (0, 0, 1)
```

**关键发现**：
- `ReferenceNormal` 通常是 `(0, 0, 1)`（Z 轴向上）
- `NormalThreshold = 0` → 只生成在完全平坦的区域
- `NormalThreshold = 90` → 生成在所有区域（禁用过滤）

---

### 4. 高度限制（Height Range）

**功能**：根据 **高度** 过滤采样点。

#### 参数配置

| 参数 | 类型 | 说明 |
|-----|------|------|
| `bFilter by Height` | `bool` | 是否启用高度过滤 |
| `Min Height` | `float` | 最小高度 |
| `Max Height` | `float` | 最大高度 |

#### 使用示例

**目标**：只在海拔 100-500 米之间生成树木。

```
[Surface Sampler]
    bFilter by Height = true
    Min Height = 100.0
    Max Height = 500.0
```

**关键发现**：
- 高度是 **世界坐标**（Z 轴）
- 适用于 **山地地形**（不同海拔不同植被）
- 可以组合 **法线过滤** + **高度限制**

## 实践案例

### 案例 1：创建自然分布的森林

**目标**：根据密度贴图和坡度，生成自然的森林。

#### 步骤 1：准备密度贴图

1. 创建 `RT_ForestDensity`（Render Target）
2. 使用 **噪声材质** 填充：
   ```
   [Perlin Noise] → [RT_ForestDensity]
   ```
3. 保存为 `T_ForestDensity.png`

#### 步骤 2：配置 Surface Sampler

```
[Surface Sampler]
    Density Texture = T_ForestDensity.png
    Texture Density Scale = 1.5
    bFilter by Normal = true
    Normal Threshold = 30.0
    bFilter by Height = true
    Min Height = 100.0
    Max Height = 800.0
```

#### 步骤 3：添加变换和生成

```
[Surface Sampler] → [Transform Points] → [Static Mesh Spawner]
```

**Transform Points**：
- `Rotation Yaw`：0-360（随机旋转）
- `Scale`：0.8-1.2（随机缩放）

**Static Mesh Spawner**：
- `Static Mesh`：`Tree_01`
- `Cull Distance`：5000

#### 步骤 4：测试

**预期结果**：森林自然分布（密度随噪声变化），避开陡坡和高海拔。

---

### 案例 2：根据地形图层生成不同植被

**目标**：在 Grass 图层生成草，在 Rock 图层生成石头。

#### 步骤 1：创建两个 PCG 图表

**图表 1**（生成草）：
```
[Surface Sampler]
    Layer Name = "Grass"
    Layer Weight Threshold = 0.5

→ [Static Mesh Spawner]
    Static Mesh = Grass_01
```

**图表 2**（生成石头）：
```
[Surface Sampler]
    Layer Name = "Rock"
    Layer Weight Threshold = 0.3

→ [Static Mesh Spawner]
    Static Mesh = Rock_01
```

#### 步骤 2：在场景中放置两个 PCG Volume

- **Volume 1**：赋值图表 1（生成草）
- **Volume 2**：赋值图表 2（生成石头）

#### 步骤 3：测试

**预期结果**：
- 草地区域生成草
- 岩石区域生成石头
- 过渡区域混合生成

## 常见错误

### Error 1：密度贴图没有效果

**症状**：设置了 `Density Texture`，但采样密度没有变化。

**原因**：
1. 纹理没有启用 `sRGB = false`
2. 纹理尺寸太小（< 512x512）
3. `Texture Density Scale` 设置错误

**解决**：
1. 在纹理编辑器中，设置 `sRGB = false`
2. 使用高分辨率纹理（1024x1024+）
3. 调整 `TextureDensityScale`（试试 0.5 或 2.0）

### Error 2：法线过滤不工作

**症状**：设置了 `Normal Threshold`，但点仍然在陡坡上生成。

**原因**：
1. `Reference Normal` 设置错误
2. 表面没有正确的法线信息（如 BSP）

**解决**：
1. 确保 `Reference Normal = (0, 0, 1)`（Z 轴向上）
2. 使用 `Landscape` 或 `Static Mesh`（有正确法线）
3. 调试：使用 `Debug Draw` 显示法线

### Error 3：高度限制不生效

**症状**：设置了 `Min Height` 和 `Max Height`，但仍然在范围外生成。

**原因**：
1. 高度值是 **世界坐标**，不是 **相对坐标**
2. 地形有 **负海拔**（如 -100 米）

**解决**：
1. 使用 `Place Actors` 工具查看当前高度
2. 调整 `Min Height` 和 `Max Height`（考虑负海拔）
3. 调试：使用 `Debug Draw` 显示点的高度
