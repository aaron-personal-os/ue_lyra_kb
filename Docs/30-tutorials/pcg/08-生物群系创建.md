# 生物群系创建

> **前置知识**：[07-实例生成器](./07-实例生成器.md)
> **预计阅读时间**：40 分钟

## 概念直觉

### 什么是"生物群系"？

**生物群系（Biome）**：具有 **相似气候、地形、植被** 的区域。

```
森林生物群系：
  - 地形：海拔 100-800m，坡度 < 30°
  - 植被：树木（密度高）、灌木（密度中）、草（密度低）
  - 气候：湿润（降水多）
```

### 为什么用 PCG 创建生物群系？

**传统方式**：
```
美术手动布置 → 耗时数周 → 修改困难
```

**PCG 方式**：
```
定义规则 → 自动生成 → 即时调整
```

## 技术机制

### 1. 地形分析

#### 高度分析（Altitude）

**原理**：根据 **海拔** 决定植被类型。

```
海拔 < 100m  → 草原
海拔 100-500m → 森林
海拔 500-1000m → 针叶林
海拔 > 1000m → 雪山
```

**实现**：
```
[Surface Sampler]
    bFilter by Height = true
    Min Height = 100
    Max Height = 500

→ [Static Mesh Spawner]
    Static Mesh = Tree_01
```

#### 坡度分析（Slope）

**原理**：根据 **坡度** 决定植被类型。

```
坡度 < 15°  → 草原（平坦）
坡度 15-30° → 森林（缓坡）
坡度 30-45° → 岩石（陡坡）
坡度 > 45°  → 无植被（悬崖）
```

**实现**：
```
[Surface Sampler]
    bFilter by Normal = true
    Normal Threshold = 30.0
    Reference Normal = (0, 0, 1)

→ [Static Mesh Spawner]
    Static Mesh = Rock_01
```

#### 湿度分析（Humidity）

**原理**：根据 **距离水源的远近** 决定植被类型。

```
距离水源 < 100m  → 湿地植物
距离水源 100-500m → 森林
距离水源 > 500m  → 干旱植物
```

**实现**：
```
[Surface Sampler]
    Density Texture = T_HumidityMap.png
    Texture Density Scale = 1.0

→ [Static Mesh Spawner]
    Static Mesh = SwampPlant_01
```

---

### 2. 多层植被生成

**概念**：一个生物群系包含 **多层植被**（从地面到树冠）。

```
第 1 层：地面覆盖（草、小花）
第 2 层：灌木丛（高度 0.5-2m）
第 3 层：小树（高度 2-10m）
第 4 层：大树（高度 > 10m）
```

**实现**：

```
[Surface Sampler 1] (Density=0.8) → [Static Mesh Spawner 1] (Grass)
[Surface Sampler 2] (Density=0.3) → [Static Mesh Spawner 2] (Bush)
[Surface Sampler 3] (Density=0.1) → [Static Mesh Spawner 3] (Small Tree)
[Surface Sampler 4] (Density=0.05) → [Static Mesh Spawner 4] (Big Tree)
```

**关键技巧**：
- 使用 **不同 Seed**（避免重叠）
- 使用 **不同高度范围**（避免穿插）
- 使用 **密度贴图**（控制分布）

---

### 3. 生态系统模拟

#### 植物竞争

**原理**：大树会 **遮挡阳光**，小草无法生长。

**实现**：
1. 先生成大树
2. 计算大树的 **阴影范围**
3. 在小树的生成逻辑中，**排除阴影范围**

```
[Surface Sampler (Big Trees)] → [Static Mesh Spawner (Big Trees)]
                                    ↓
                            [Create Shadow Mask]
                                    ↓
[Surface Sampler (Small Trees)] → [Filter by Shadow Mask] → [Static Mesh Spawner (Small Trees)]
```

#### 植物共生

**原理**：某些植物 **喜欢** 生长在大树旁边（如蘑菇）。

**实现**：
1. 生成大树
2. 在大树 **周围** 生成蘑菇（距离 < 2m）

```
[Surface Sampler (Big Trees)] → [Static Mesh Spawner (Big Trees)]
                                    ↓
                            [Create Points Around Trees]
                                    ↓
[Surface Sampler (Mushrooms)] → [Static Mesh Spawner (Mushrooms)]
```

## 实践案例

### 案例 1：创建森林生物群系

**目标**：创建一个多层的森林生物群系。

#### 步骤 1：地形分析

**高度范围**：100-800m
**坡度范围**：< 30°

```
[Surface Sampler]
    bFilter by Height = true
    Min Height = 100
    Max Height = 800
    bFilter by Normal = true
    Normal Threshold = 30.0
```

#### 步骤 2：多层植被

**第 1 层：草**
```
[Surface Sampler 1]
    Density = 0.8
    Seed = 123

→ [Transform Points]
    Scale Min = (0.5, 0.5, 0.5)
    Scale Max = (1.0, 1.0, 1.0)

→ [Static Mesh Spawner]
    Static Mesh = Grass_01
```

**第 2 层：灌木**
```
[Surface Sampler 2]
    Density = 0.3
    Seed = 456

→ [Transform Points]
    Scale Min = (0.8, 0.8, 0.8)
    Scale Max = (1.2, 1.2, 1.2)

→ [Static Mesh Spawner]
    Static Mesh = Bush_01
```

**第 3 层：树木**
```
[Surface Sampler 3]
    Density = 0.1
    Seed = 789

→ [Transform Points]
    Scale Min = (0.8, 0.8, 0.8)
    Scale Max = (1.5, 1.5, 1.5)
    Rotation Yaw = 0-360

→ [Static Mesh Spawner]
    Static Mesh = Tree_01
```

#### 步骤 3：测试

**预期结果**：
- 地面覆盖草
- 中间层有灌木
- 顶层有树木
- 自然分布，无重叠

---

### 案例 2：创建沙漠生物群系

**目标**：创建一个干旱的沙漠生物群系。

#### 步骤 1：地形分析

**高度范围**：0-500m
**坡度范围**：< 15°（沙漠通常平坦）

```
[Surface Sampler]
    bFilter by Height = true
    Min Height = 0
    Max Height = 500
    bFilter by Normal = true
    Normal Threshold = 15.0
```

#### 步骤 2：植被配置

**主要植被**：仙人掌（密度低）

```
[Surface Sampler]
    Density = 0.02 (密度很低)
    Seed = 123

→ [Transform Points]
    Scale Min = (0.8, 0.8, 1.0)
    Scale Max = (1.2, 1.2, 1.5)

→ [Static Mesh Spawner]
    Static Mesh = Cactus_01
```

**次要植被**：干草（密度中）

```
[Surface Sampler]
    Density = 0.1
    Seed = 456

→ [Transform Points]
    Scale Min = (0.5, 0.5, 0.5)
    Scale Max = (0.8, 0.8, 0.8)

→ [Static Mesh Spawner]
    Static Mesh = DryGrass_01
```

#### 步骤 3：添加细节

**地面细节**：石头

```
[Surface Sampler]
    Density = 0.05
    Seed = 789

→ [Transform Points]
    Scale Min = (0.3, 0.3, 0.3)
    Scale Max = (0.8, 0.8, 0.8)

→ [Static Mesh Spawner]
    Static Mesh = Rock_01
```

#### 步骤 4：测试

**预期结果**：
- 平坦的沙漠地面
- 稀疏的仙人掌
- 干枯的草
- 散落的石头

## 常见错误

### Error 1：生物群系之间过渡不自然

**症状**：森林和沙漠之间有明显边界。

**原因**：没有使用 **渐变过渡**。

**解决**：
1. 使用 **密度贴图**（控制过渡区域）
2. 使用 **多层生物群系**（混合区域）
3. 使用 **插值**（平滑过渡）

### Error 2：植被重叠（穿插）

**症状**：树木和灌木穿插在一起。

**原因**：没有使用 **不同高度范围** 或 **不同 Seed**。

**解决**：
1. 使用 **不同 Seed**（避免相同位置）
2. 使用 **高度限制**（如树木 > 2m，灌木 < 2m）
3. 使用 **碰撞检测**（避免重叠）

### Error 3：性能爆炸（FPS 暴跌）

**症状**：生成生物群系后，FPS 从 60 降到 10。

**原因**：
1. 点数量过多（>50000）
2. 没有使用 **HISM**（每个都是独立 Actor）
3. **剔除距离** 设置过大

**解决**：
1. 降低 `Density`
2. 确保 `bUseHISM = true`
3. 设置合理的 `Cull Distance`（如 5000）
4. 使用 **分块生成**（多个 PCG Volume）

## 总结

通过本篇你学到了：

1. **生物群系概念** — 不同区域有不同植被（森林、沙漠、雪山），通过权重贴图控制
2. **多层生成** — 使用多个 Surface Sampler + 不同 Seed，避免植被穿插
3. **性能优化** — 使用 HISM、分块生成（多个 PCG Volume）、合理剔除距离
4. **常见问题** — 植被穿插（不同高度/Seed）、性能爆炸（降低密度、启用 HISM）
